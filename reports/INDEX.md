## 📝 Quick Reference

### Installation One-Liner
```bash
cd ~ && git clone https://github.com/bsf2608/cavicapture.git && cd cavicapture && \
python3 -m venv env && source env/bin/activate && \
pip install opencv-python-headless numpy pillow RPi.GPIO picamera2
```

### First Capture
```bash
cp example_config.ini test.ini
# Edit test.ini (set output_dir, GPIO_light_channel)
sudo python3 cavicapture.py --config test.ini --preview
```

### First Processing
```bash
python3 caviprocess.py --config test.ini
```

### View Results
```bash
sqlite3 ./captures/*/capture.db "SELECT * FROM captures LIMIT 5;"
```
